    - 1️⃣ What is the difference between var, let, and const?
    Ans: 

    var:  Function-scoped, can be redeclared & reassigned, hoisted (initialized as undefined).

    let:  Block-scoped, cannot be redeclared, can be reassigned, hoisted but in temporal dead zone.

    const:  Block-scoped, cannot be redeclared or reassigned, hoisted but in temporal dead zone; objects/arrays can be mutated.

    - 2️⃣ What is the spread operator (...)?

    Ans: The spread operator (...) in JavaScript is used to expand an iterable (like an array or object) into individual elements.

    - 3️⃣ What is the difference between map(), filter(), and forEach()?

    Ans: map() → Creates a new array by transforming each element.

        filter() → Creates a new array with elements that pass a condition.

        forEach() → Does not return anything, just executes a function on each element.

    - 4️⃣ What is an arrow function?

    Ans: An arrow function is a compact way to write functions in JavaScript using the => syntax. It is shorter than the traditional function keyword and is commonly used for small operations or callbacks.

    - 5️⃣ What are template literals?

    Ans: Template literals are a way to create strings in JavaScript using backticks ` instead of quotes. They allow multi-line strings and embedding expressions easily.
